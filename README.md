# Weather Dashboard Application

## Overview
A responsive weather dashboard application built using JavaScript and OpenWeatherMap API. It shows current weather, a 5-day forecast, and saves user preferences using Local Storage.

## Features
- Search weather by city
- Current weather details
- 5-day forecast
- Debounced search input
- Geolocation support
- Local Storage persistence
- Error handling and loading states
- Responsive and accessible UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6 Modules)
- OpenWeatherMap API

## Project Structure
index.html  
css/styles.css  
js/app.js  
js/api.js  
js/storage.js  

## How to Run
1. Clone the repository
2. Replace `YOUR_OPENWEATHERMAP_API_KEY` in `api.js`
3. Open `index.html` in browser
